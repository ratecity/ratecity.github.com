Parameters

A number of parameters are required to retrieve results.

cl_debug
Enable debug mode.
cl_dist_id
an integer containing your partner code, supplied by RateCity
cl_load_jquery
a boolean that determines whether the widget should load jquery or not. If you use jquery already, you should set this to false to avoid version collisions. If you disable this, please let us know.
