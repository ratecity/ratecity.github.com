RateCity Widgets

Bring RateCity top products to your website.

Introduction

The purpose of this document is to define consumption of the RateCity (http://www.ratecity.com.au) JS Widget API. You are assumed to be familiar with HTML and JavaScript programming, along with CSS (Cascading Style Sheets) to control the style.


Getting it working

Embedding the code

Firstly, embed the following code on the page, and use the parameter list below to fill out the 'xx' parameters.

Parameters

A number of parameters are required to retrieve results.

partner
a string containiner your partner code, supplied by RateCity
feed
a string containing your feed identifier, supplied by RateCity
title
a string containing the optional feed title
num
an integer describing the number of products you wish to display
fields
an array of string, containing the columns you wish to render from the feed
direct_apply
always false, setting to true enables apply click from the whole table row
cssTable
a string reference to the css when rendering the table, see below
applyText
a string containing the text displayed on the apply link, set to empty string if you are using images for apply links

Customised Styles

Most of the classes are self-explanatory, but are detailed below:

rc-toptable
The outer div for the widget
ratecity_top
Identifies the outer div for the table title.
ratecity_tablehead
Identifies the heading (h2) used for the table title.
ratecity_header
Identifies the table row containing the data column headers
ratecity_rowodd and ratecity_roweven
Identifies the table row containing product data
ratecity_company
Identifies the table data element containing the company logo.
See above for the markup within this element
ratecity_product
Identifies the table data element containing product description.
See above for the markup within this element
ratecity_best_rate_percentage
Identifies an optional column added using the 'fields' parameter, in this case best_rate_percentage
ratecity_apply
Identifies the table data element containing the markup for a product application.
See above for the markup within this element
ratecity_footer
Outer div for the footer.
See above for the markup within this element
